# Feature Progress Tracker

## Completed Features
- [x] Basic HTML/CSS/JS scaffolding.
- [x] Fast local static server using Bun.
- [x] SVG parsing and 3D extrusion via Three.js.
- [x] Applied hardcoded gold material with reflective lighting.
- [x] Horizontal spinning animation loop.
- [x] GLB file export functionality.
- [x] Support for importing 1 SVG at a time.
- [x] UI color picker to change material color dynamically with local storage persistence.
- [x] UI depth input and automatic depth calculation based on SVG dimensions.
- [x] Advanced invisible path filtering (ignores display="none", hidden, and white backgrounds).
- [x] HD Quality toggle to vastly increase geometry curve density for smoother paths.
